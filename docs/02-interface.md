# 02. 界面概览

```
┌─ 工具栏 ───────────────────────────────────────────┐
│ t1.kln *    📄新建 📂打开 💾保存 💾另存 📦 ⚡生成 👁 │
├──────────┬─────────────────────────────────────────┤
│ 📦 节点  │                                         │
│          │          画布 (ReactFlow)                │
│ ▸ 消息提醒│                                         │
│ ▸ MultiTTS│                                         │
│ ▸ NamePick│         [节点] ──→ [节点]               │
│ ▸ Everythi│                                         │
│ ▸ 系统监控│                                         │
│ ▸ ClassIsl│                                         │
│          │                                         │
│ 🔌 协议   │                                         │
│ 💻 Progra│                                         │
├──────────┴─────────────────────────────────────────┤
│ 生成的 Python 代码（可折叠）                       │
└────────────────────────────────────────────────────┘
```

## 区域说明

| 区域 | 位置 | 功能 |
|------|------|------|
| **工具栏** | 顶部 | 文件名、新建/打开/保存、生成代码 |
| **节点面板** | 左侧 240px | 可折叠的分类节点列表，拖拽添加 |
| **画布** | 中央 | 节点编排、连线、缩放、平移 |
| **代码面板** | 底部 | 生成的 Python 代码，可折叠 |

## 文件状态

- 文件名后 `*` 表示有未保存的修改
- 保存后 `*` 消失

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建 |
| `Ctrl+O` | 打开 |
| `Ctrl+S` | 保存 |
| `Ctrl+G` | 生成代码 |
| `Ctrl+`` | 切换代码面板 |
| `Delete` | 删除选中节点 |
